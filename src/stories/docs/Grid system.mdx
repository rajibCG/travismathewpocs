# Grid system

We leverage Tailwind's CSS grid for column alignment -- however, we want to use Tailwind ONLY for the grid system, and not use Tailwind for anything else on this project (everything else should be custom CSS code)

Tailwind's CSS grid system allows us to create responsive and well-structured layouts effortlessly.
With just a few utility classNamees, we can craft complex grid structures. The grid system is built on top of CSS Grid, a powerful native browser feature that allows for precise control over grid layouts.

Check out the [Tailwind CSS documentation ](https://tailwindcss.com/docs/grid-template-columns) for grid-template-columns
